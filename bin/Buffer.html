Class Buffer

 
java.lang.Object
  Buffer
public abstract class Buffer
extends java.lang.Object

releasePagePolicy

 
public abstract int releasePagePolicy(java.nio.channels.FileChannel fc, int blockNumber)

Method defining the policy for releasing pages in the buffer. This method is invoked when it is required to load a block into a page and there are no empty pages in the buffer. The method is used for deciding which page is going to be released and then used for loading that newly requested block.

Parameters:

fc  reference to the file (to which the block to be read belongs).

blockNumber the block to be read.

Returns:

The released page identifier.

referencedPage

 
public abstract void referencedPage(int i)

This method is invoked anytime a page (i) is accessed. For implementing some certain release-page policies (such as least recently used) it is required to know when each page was last accessed, how often it is accessed, etc. Thus, this method is useful for (jointly with the former method, releasePagePolicy) implementing some certain release page policies.

Parameters:

i  page that has been accessed.
